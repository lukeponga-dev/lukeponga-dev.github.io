## ğŸš€ Getting Started

### **Prerequisites**
- Node.js 16+ (for local development)
- Git
- A modern web browser

### **Local Development**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lukeponga-dev/lukeponga-dev.github.io.git
   cd lukeponga-dev.github.io
   ```

2. **Open locally:**
   ```bash
   # Simple way - open index.html directly
   open index.html
   
   # Or use a local server (Python)
   python3 -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **For development with Node:**
   ```bash
   npm install
   npm run dev
   ```

### **Making Changes**

1. Edit `index.html` for structure changes
2. Update `css/styles.css` for styling
3. Modify `js/script.js` for functionality
4. Add project images to `/images/` folder
5. Test locally before pushing

In `css/styles.css`, modify the CSS variables:
```css
:root {
  --primary-color: #3b82f6;      /* Change primary color */
  --dark-bg: #0c121e;             /* Change background */
```

## ğŸ› Troubleshooting

### **Images Not Loading**
- Ensure image file names match exactly
- Check image paths use `/images/filename`
- Verify images exist in the `/images/` folder

### **Dark Mode Not Working**
- Check browser localStorage is enabled
- Clear browser cache and reload
- Verify `localStorage.getItem('theme')` in console

### **Layout Toggle Not Working**
- Ensure JavaScript file is loaded correctly
- Check browser console for errors
- Verify HTML IDs match JavaScript references
